پلاگین مستقل رزرو نوبت آنلاین برای سالن، باشگاه، مشاوره حقوقی، آموزشگاه، کلینیک زیبایی و هر کسب‌وکار محلی نوبت‌محور — با تقویم جلالی، پیامک و درگاه پرداخت ایرانی از روز اول و بدون add-on پولی جدا.

== توضیحات ==

نوبتیار یک سیستم رزرو نوبت کامل برای وردپرس است که با شورت‌کد `[nobatyar_booking]` در هر صفحه یا نوشته قابل استفاده است. برخلاف بسیاری از پلاگین‌های مشابه (Bookly، Booknetic)، نوبتیار هیچ فیچر حیاتی را پشت add-on پولی جداگانه قرار نمی‌دهد: پیامک و درگاه پرداخت همان لایسنس اصلی شامل می‌شود.

= امکانات اصلی =

* **موتور رزرو** با محاسبه اسلات‌های واقعی (نه فقط سر ساعت)، امکان رزرو دسته‌ای، و جلوگیری از تداخل زمانی بین نوبت‌های یک سرویس‌دهنده
* **تقویم جلالی به‌عنوان شهروند درجه‌یک** — تمام نمایش‌های ادمین و فرانت بر اساس تاریخ شمسی، در حالی که ذخیره‌سازی پایگاه‌داده همواره میلادی است
* **اطلاع‌رسانی پیامکی** با پشتیبانی از کاوه‌نگار و ملی‌پیامک (معماری مبتنی بر اینترفیس برای افزودن سرویس‌دهنده جدید بدون تغییر در core)
* **درگاه‌های پرداخت ایرانی**: زرین‌پال، آیدی‌پی، نکست‌پی
* **سامانه لایسنس** با دوره Grace 14 روزه (بدون حذف داده در صورت انقضا) و امکان انتقال لایسنس به دامنه جدید بدون نیاز به تماس با پشتیبانی
* **پنل مدیریت کامل**: لیست نوبت‌ها با فیلتر و تغییر وضعیت، نمایش تقویمی ماهانه، گزارش‌گیری درآمد و وضعیت نوبت‌ها، و ویزارد تنظیمات فارسی ۵ مرحله‌ای (اصطلاحات، پیامک، پرداخت، لایسنس، خلاصه)
* **اصطلاحات قابل تغییر** — می‌توانید "سرویس‌دهنده" را به "آرایشگر"، "استایلیست"، "مشاور" و غیره تغییر دهید بدون نیاز به ترجمه دستی
* **بارگذاری شرطی منابع** — کدهای JS/CSS فقط در صفحات حاوی شورت‌کد یا صفحات مدیریت پلاگین بار می‌شوند

= REST API =

نوبتیار یک REST API کامل زیر namespace `nobatyar/v1` ارائه می‌دهد (`/availability`، `/bookings`، `/payments/init`، `/payments/callback`، `/license/activate`، `/license/status`) به همراه action hooks (`nobatyar_booking_created`, `nobatyar_booking_status_changed`, `nobatyar_payment_verified`, `nobatyar_license_status_changed`) و فیلترها (`nobatyar_available_slots`, `nobatyar_terminology_label`) برای توسعه‌دهندگان.

== نصب ==

1. پوشه `nobatyar-booking` را در `wp-content/plugins/` قرار دهید (یا از طریق صفحه افزونه‌های وردپرس آپلود کنید)
2. پلاگین را از منوی «افزونه‌ها» فعال کنید
3. از منوی «نوبتیار» در پیشخوان، گام‌به‌گام ویزارد تنظیمات (اصطلاحات، پیامک، پرداخت، لایسنس) را تکمیل کنید
4. سرویس‌دهنده‌ها و خدمات خود را تعریف کرده و شورت‌کد `[nobatyar_booking]` را در هر صفحه‌ای قرار دهید

== سوالات متداول ==

= آیا برای ارسال پیامک باید add-on جداگانه بخرم؟ =

خیر. ارسال پیامک با کاوه‌نگار یا ملی‌پیامک بخشی از همان لایسنس اصلی است.

= آیا تقویم نمایش‌داده‌شده میلادی است یا شمسی؟ =

تمام رابط کاربری (فرانت و پیشخوان) شمسی است؛ تاریخ‌ها فقط در لایه دیتابیس به‌صورت میلادی ذخیره می‌شوند.

= اگر لایسنس منقضی شود داده‌هایم حذف می‌شوند؟ =

خیر. پس از انقضا یک دوره Grace ۱۴ روزه فعال می‌شود (پیامک/پرداخت غیرفعال، رزرو پایه فعال) و حتی پس از پایان Grace هیچ داده‌ای حذف نمی‌شود.
